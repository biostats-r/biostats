Introduction

Data analysis are often not done in one step, but there are often several rounds where the data, data analysis and/or figures need to be updated. If data is added or a mistake has been made, all the scripts have to be run again to update the analysis, figures and output. This normally takes several rounds and for computational heavy analysis, it can become time consuming.

Targets is a pipeline tool for statistics and data science in R. It takes care of the dependencies of the whole workflow and only updates the steps that are outdated, but skips those where the upstream code has not been changed. Targets therefore maintains reproducible workflows and avoids unnecessary repetition and can save costly running time.

When is targets useful?

  • When the code has a long runtime because it is slow or complex

  • When the workflow has interconnected tasks with dependencies

  • When there are different outputs (e.g. presentation and report)

Before you start

You will need to have installed

  • RStudio (version 2022.2 or newer)
  • R
  • quarto
  • quarto R package (install.packages("quarto"))
  • targets R package (install.packages("targets"))
  • tarchetypes R package (install.packages("tarchetypes"))

If you cannot install quarto on your computer (perhaps because of permission problems), either use the Rmarkdown tutorial, or use rstudio.cloud.
